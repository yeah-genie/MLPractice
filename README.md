# 일상 해부학 | Everyday Anatomy

> 격주로 일상 속 신박한 궁금증을 데이터로 풀어내는 인스타그램 프로젝트.  
> 누구나 하는 타이타닉·주식 분석이 아닌, 기획력과 실행력을 동시에 보여주는 포트폴리오.

---

## 왜 이 프로젝트인가

데이터 분석 직무에서 중요한 건 '데이터를 어떻게 가공하는가'만이 아니다.  
**데이터를 통해 어떤 인사이트를 도출하고, 대중에게 설득력 있게 전달하는가(데이터 스토리텔링)**가 점점 더 중요해지고 있다.

이 프로젝트는 두 채널로 운영한다:

| 채널 | 역할 |
|------|------|
| 인스타그램 | 직관적인 차트 + 한 줄 인사이트 (대중용) |
| GitHub + 기술 블로그 | 데이터 수집·전처리·분석 전 과정 기록 (포트폴리오용) |

블로그에 **개인정보 비식별화, 데이터 윤리·거버넌스**에 대한 고민까지 남기면 실무형 인재로 어필 가능.

---

## 계정 컨셉

### 컨셉 A — 데이터 탐정소 (Data Detective Agency)
- **무드:** 일상 속 소소한 미스터리를 데이터 돋보기로 파헤치는 사립 탐정
- **디자인:** 사건 파일, 증거 보드(String board) 스타일 피드
- **말투:** "오늘 의뢰가 들어온 사건은...", "증거 데이터를 수집한 결과..."

### 컨셉 B — 일상 해부학 (Everyday Anatomy) ← 추천
- **무드:** 감성적인 일상 현상을 뼛속까지 이성적인 숫자로 건조하게 해부하는 '이과적 감성'
- **디자인:** 블루프린트·실험 보고서 느낌의 미니멀 디자인
- **특징:** 분석 결과에 독자적인 지표명을 붙임 → 예: "한국 드라마 평균 키스 도달 지수(KDI) = 6.5회"  
  자체 네이밍은 공유·바이럴에 특히 유리함

## 분석 주제 8선

### ① 동의합니다를 누른 개인정보, 다 읽으려면 며칠?
한국인이 가장 많이 쓰는 앱 상위 50개의 개인정보처리방침·이용약관 텍스트를 분석.  
평균 읽기 시간 계산 + '제3자 제공', '마케팅 활용' 같은 독소조항 키워드 빈도 순위.

- **데이터:** 구글 플레이스토어·앱스토어 인기 앱 약관 페이지 (BeautifulSoup 크롤링)
- **분석:** 텍스트 길이, 독소조항 키워드 빈도, 가독성 지수
- **Power BI:** 앱별 가로 막대 차트, 키워드 히트맵

---

### ② 수학 문제 난이도와 여백 면적의 상관관계
어려운 문제일수록 풀이 공간이 많을까, 텍스트가 빽빽할까?  
수능 기출 PDF → 텍스트 픽셀 vs 여백 픽셀 비율 → 실제 오답률과 상관관계 분석.

- **데이터:** EBSi 수능·모의고사 기출 PDF + 문항별 정답률 통계
- **분석:** PDF → 이미지 변환, OpenCV 여백 픽셀 계산, 피어슨 상관계수
- **Power BI:** 산점도 (여백 비율 vs 오답률), 회귀 추세선

---

### ③ 힐링 게임 속 노동 강도: 현실 직장인 vs 게임 농장주
스타듀밸리·모동숲 캐릭터의 일일 행동량을 현실 칼로리·노동 시간으로 환산.  
"이 게임은 진짜 힐링일까, 중노동일까?"

- **데이터:** Fandom 위키 (아이템 채집 시간, 맵 크기, 이동 속도 등 수치)
- **분석:** 이동 거리→칼로리 환산, 노동 시간 계산, 최저임금 대비 시급 산출
- **Power BI:** 인포그래픽 스타일 카드, 직업별 비교 막대

---

### ④ 로맨스 드라마 첫 키스, 언제·어디서 터질까
인기 로맨스 드라마 30편 기준, 첫 키스신의 평균 회차·분수·장소·날씨 통계.  
**한국 드라마 키스 도달 지수(KDI)** 를 자체 정의.

- **데이터:** OTT 자막 파일(SRT/SMI) 텍스트 마이닝, 나무위키 줄거리
- **분석:** 키워드('키스', '입술') 타임스탬프 추출, 회차별 분포, 장소 분류
- **Power BI:** 히스토그램, 도넛 차트 (장소 비율), 타임라인 산점도

---

### ⑤ 내가 마신 커피 한 잔, 몇 개의 기업이 알고 있을까
하루 스마트폰·브라우저 사용 중 내 일상을 추적하는 제3자 트래커 수 시각화.  
배너 광고 하나 뒤에 숨겨진 수십 개의 수집기를 드러내는 시각적 충격.

- **데이터:** Ghostery·Privacy Badger 확장 프로그램 CSV 내보내기, GDPR 데이터 아카이브
- **분석:** 트래커 도메인 분류, 기업별 집계, 시간대별 추적 밀도
- **Power BI:** 네트워크 다이어그램, 시간대별 꺾은선

---

### ⑥ 틀리는 데에도 공식이 있다: 오답 패턴 분석
수학·논리 문제에서 '몰라서' 틀리는 게 아니라 특정 함정에 걸려 틀리는 패턴 분류.  
가장 빈번한 오답 유형과 비율 시각화.

- **데이터:** Kaggle EdNet 데이터셋 (학생 문제 풀이 로그), GSM8K 오답 케이스
- **분석:** 오답 유형 클러스터링, 빈도 분석, 문항 속성 교차분석
- **Power BI:** 트리맵, 유형별 누적 막대

---

### ⑦ 노동요의 정석: BPM이 뇌의 능률을 지배할까
'코딩할 때 듣는 플리', '집중력 로파이' 플레이리스트의 음악 특성 분석.  
실제로 특정 BPM·에너지 수치의 곡들이 집중 플리에 집중되어 있는지 검증.

- **데이터:** Spotify Web API — Audio Features (Tempo, Energy, Valence, Acousticness)
- **분석:** 플리 유형별 특성 분포, ANOVA 검정, 최적 BPM 구간 도출
- **Power BI:** 바이올린 플롯(Python 시각화), 레이더 차트

---

### ⑧ 배달 앱 '신라면 맵기'는 진짜 신라면 맵기일까
맵기 리뷰 텍스트 감성 분석으로 프랜차이즈별 맵기 인플레이션 현상 측정.  
"신라면 맵기라더니 엽떡 수준" 같은 텍스트 비율로 브랜드별 맵기 신뢰도 순위.

- **데이터:** 네이버 플레이스·배달 앱 리뷰 중 '신라면', '맵기' 포함 텍스트 (robots.txt 준수)
- **분석:** 한국어 감성 분석, 맵기 기대-현실 괴리 지수 산출
- **Power BI:** 브랜드별 점수 막대, 워드클라우드

## 실행 순서 (현재 구현: ⑦ 유튜버 휴재 패턴 분석)

```bash
pip install -r requirements.txt

python collect.py              # ① 영상 메타데이터 수집
python clean.py                # ② 분석 지표 생성 → data/cleaned/*_videos.csv
python analyze_transcripts.py  # ③ 자막 키워드 분석 → data/cleaned/*_transcripts.csv
```

> `*_videos.csv` 와 `*_transcripts.csv` 는 **video_id** 키로 Power BI에서 관계 연결.

---

## 파일 구조

```
collect.py              유튜브 채널 데이터 수집
clean.py                분석 지표 생성 (Z-score, 위험 점수 등)
analyze_transcripts.py  자막 키워드 분석 (번아웃·사과·불확실 신호)
data/raw/               수집 원본 (gitignore)
data/cleaned/           Power BI 임포트용 CSV
.env                    API 키 (gitignore — 절대 공유 금지)
```
